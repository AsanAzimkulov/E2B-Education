Если есть файлы с описанием изменений, то они взяты с папки главной страницы, то есть если описанное там, уже сделано, то повторяться не нужно.

Если тени других блоков обрезаются, то со всех страниц верстки, в файле page.scss убрать блок кода - 

    // &>section:not(.breadcrumbs){
    //   max-width: 100vw !important;
    //   overflow-x: hidden !important;
    //   overflow-y: visible;
    // }

Если будет на мобилках листаться влево вправо, то ко всем блокам-слайдерам, в стили блока добавить блок кода - 

  @at-root{
    section#{&}{
      max-width: 100vw;
      overflow-x: hidden !important;
      max-width: 100vw !important;
    }
  }

Например: 
.block{

 @at-root{
    section#{&}{
      max-width: 100vw;
      overflow-x: hidden !important;
      max-width: 100vw !important;
    }
  }

}
